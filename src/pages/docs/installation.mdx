## Command line interface

```shell
yarn mh init
```

## Manual installation

```shell
yarn add @molehill-ui/babel-plugin @molehill-ui/postcss-plugin -D
npm install @molehill-ui/babel-plugin @molehill-ui/postcss-plugin --save-dev
```

### Add the Babel plugin

The babel plugin should be added to the array of plugins before your CSS-in-JS library.

```js
// .babelrc
{
  "plugins": ["@molehill-ui"]
}
```

### Add the PostCSS plugin

```js
// postcss.config.js
module.exports = {
  plugins: [
    '@molehill-ui/postcss-plugin',
  ],
}
```

### Create the config file

```js
// molehill.config.js
const { createCssTheme } = require('@molehill-ui/theme')

const theme = createCssTheme()

module.exports = theme
```

### Include MoleHill UI in your CSS

```css
/* ./your-top-level-css-file.css */
@import 'molehill-ui/variables';
@import 'molehill-ui/base';
```
